百度翻译GoldenDict插件帮助文档

---

常规使用

正常使用时，将需要翻译的非中文字（词）或一段文字（词）输入，即可自动识别并翻译为中文，如果输入为中文，则会输出英文翻译。

强制源语言类型与目标语言类型

- 使用如下方式来指定源语言类型a与目标语言类型b
      [a->b]:待翻译词语

- 使用如下方式来指定源语言类型a
      [a->?]:待翻译词语
- 使用如下方式来指定目标语言类型b
      [?->b]:待翻译词语
